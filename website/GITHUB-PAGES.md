# Deploy Ougi site with GitHub Pages (free)

Upload **all files** inside `website/public` (not just one page):

- `index.html` · `features.html` · `pricing.html` · `pay.html` · `access.html`
- `styles.css` · `theme.js` · `site.js` · `pay.js` · `app.js` · `config.js`

## Before upload
Edit `config.js`:
1. Set `notifyEmail` to your email
2. Prices / crypto addresses are already filled

## GitHub Pages
1. Create repo `ougi-web`
2. Upload the files above
3. Settings → Pages → Deploy from branch → `main` / root
4. Link: `https://YOUR_USERNAME.github.io/ougi-web/`

Theme toggle: Dark pink ↔ Light pink (saved in the browser).
Payment page: Card request + Crypto (BTC / ETH / USDT / SOL / LTC) with copy buttons.
