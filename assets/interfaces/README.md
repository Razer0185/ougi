# Interface preview images

Drop one image per color template here. Filenames must match:

| File | Color |
|------|--------|
| `red.png` | Red |
| `black.png` | Black |
| `white.png` | White |
| `pink.png` | Pink |
| `blue.png` | Blue |
| `purple.png` | Purple |
| `green.png` | Green |
| `orange.png` | Orange |
| `cyan.png` | Cyan |
| `gold.png` | Gold |

Also accepts `.jpg`, `.jpeg`, `.webp`, `.gif`.

Until you add a file, Ougi shows a **color-name placeholder** in the Interfaces gallery.
Send the pictures in chat and I’ll place them here for you.
